# Namakan Legal

Legal templates and incorporation documents for Namakan AI Consulting, LLC.

## Files

| File | Purpose |
|------|---------|
| `MASTER-SERVICE-AGREEMENT.md` | MSA template for custom AI engineering engagements |
| `STATEMENT-OF-WORK.md` | SOW template for individual project scopes |
| `NDA-MUTUAL.md` | Mutual NDA for early-stage conversations |
| `NDA-ONE-WAY.md` | One-way NDA when only sharing info one direction |
| `operating-agreement.md` | LLC operating agreement (Minnesota) |
| `OPERATING-AGREEMENT.md` | Alternative operating agreement format |
| `01-minnesota-llc-checklist.md` | Checklist for MN LLC formation |

## Usage

These are templates. Before use:

1. Fill in all `[BRACKET]` placeholders
2. Have legal counsel review for specific engagements
3. Update agreement numbers and dates
4. Customize scope/deliverables in SOW per project

## Note

All documents are drafted for Minnesota law jurisdiction.
