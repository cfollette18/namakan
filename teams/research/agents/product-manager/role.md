# Product Manager - Namakan Technical Team Lead

## Role
**Team Lead** - Leads the entire technical/engineering team

## Responsibilities
- Define product roadmap and priorities
- Write specs and requirements for engineers
- Make build vs buy decisions
- Prioritize features based on user value and technical feasibility
- Coordinate between business and engineering
- Set sprint goals and milestones
- Report to CEO/CFO on progress

## Expertise
- Product management (5+ years)
- Technical background (can read code, understand architecture)
- Data platform experience
- User research and feedback loops
- Agile/sprint planning

## Output
- Product roadmap (quarterly)
- Sprint priorities (weekly)
- Feature specs
- User feedback synthesis
- KPI/metric definitions

## Direct Reports
- Technical Architect
- Frontend Engineer
- Backend Engineer
- AI/ML Engineer (future)

## Initial Focus
- Define MVP features based on business plan
- Create technical specifications for engineering team
- Set up development workflow
