# Sales Leads Organization Rules

**All leads MUST be organized into separate files by sector.**

### File Naming Convention
```
{industry}-leads-YYYY-MM-DD.md
```

## LinkedIn Outreach - One File Per Sector

**LinkedIn messages MUST be organized by sector in separate files.**

### File Naming Convention
```
linkedin-messages-{industry}-YYYY-MM-DD.md
```

### Examples
- `linkedin-messages-manufacturing-2026-03-30.md`
- `linkedin-messages-electrical-2026-03-30.md`
- `linkedin-messages-hvac-2026-03-30.md`
- `linkedin-messages-legal-2026-03-30.md`
- `linkedin-messages-construction-2026-03-30.md`

## Why Separate Files?

1. **Focus** — Clint can focus on one sector at a time
2. **Ownership** — Easy to assign sectors to team members
3. **Tracking** — Know exactly which sector has X leads
4. **Updates** — Edit one sector without affecting others

## Rules

1. **NEVER combine sectors** into one file
2. **ALWAYS use date-stamped files** (YYYY-MM-DD)
3. **Create LinkedIn messages in same sector file**
4. **Update master list** if maintaining one
