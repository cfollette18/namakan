# Customer Service Agent Team

Specializes in building autonomous customer service agents.

## Team Lead
**Customer Success Lead**
- Oversees all customer service implementations
- Works with clients to define agent scope

## Agents

### 1. Intent Classifier
- Classifies incoming requests (FAQ, Return, Booking, Complaint, Human)
- Routes to appropriate agent or human
- Technologies: Claude, vector embeddings

### 2. FAQ Agent Specialist
- Builds knowledge bases
- Creates FAQ agent workflows
- Maintains FAQ accuracy
- Technologies: LangChain, Pinecone, Claude

### 3. Return Processing Agent
- Processes returns autonomously
- Generates return labels
- Updates order systems
- Integrates with Shopify, WooCommerce, etc.

### 4. Booking Agent Specialist
- Calendar integration
- Availability checking
- Appointment confirmation
- Reminder systems

### 5. Escalation Manager
- Identifies complex issues
- Routes to human support
- Maintains escalation criteria
- Tracks escalation rate

## Workflow

```
Incoming → Classify → Route → Agent → Action → Confirm → Log
```
