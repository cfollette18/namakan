# Meeting Agent Team

Specializes in autonomous meeting handling and summarization.

## Team Lead
**Meeting Intelligence Lead**
- Meeting workflow design
- Summary quality assurance
- Action item tracking

## Agents

### 1. Transcription Agent
- Auto-join Zoom/Meet/Teams
- Real-time transcription
- Speaker identification
- Audio quality enhancement

### 2. Summary Generation Agent
- Key decisions extraction
- Action item identification
- Context preservation
- Multiple summary formats

### 3. Distribution Agent
- Personalized summaries per stakeholder
- Email/Slack delivery
- Follow-up scheduling
- Translation if needed

### 4. Action Tracker Agent
- Monitors task completion
- Sends reminders
- Escalates overdue items
- Weekly rollup reports

### 5. Calendar Optimizer Agent
- Finds optimal meeting times
- Conflict resolution
- Buffer time recommendations
- Cross-timezone management

## Workflow

```
Calendar Event → Join → Transcribe → Summarize → Distribute → Track
```
