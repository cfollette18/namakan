# Logo - Namakan Wordmark (Concept 4)

**Status:** APPROVED — 2026-03-29

## Prompt Used

```
Logo design for Namakan AI company.

Design: A clean modern logo where the geometric N symbol is followed by the text "NAMAKAN" in the same teal color.

The geometric N symbol:
- Two parallel diagonal bars with rounded corners, inclined at a 45-degree angle
- Both bars start at the top and end at the bottom of the symbol
- Between those bars, two identical V-shaped bars with rounded corners form the interior part of the letter N
- One V-shaped bar starts at the top and points downward, the other starts at the bottom and points upward
- All in teal color

The text "NAMAKAN":
- Clean modern sans-serif font (like Inter or Helvetica)
- Same teal color as the symbol
- Uppercase
- Positioned to the right of the N symbol, on the same baseline

Layout: [Geometric N] NAMAKAN — all elements teal, horizontally aligned

Colors: 
- All elements: Teal (#0D9488)
- Background: White (#FFFFFF)

Style: Minimal, professional, trustworthy. Like Vercel or Linear logos. No gradients. Flat design.
```
